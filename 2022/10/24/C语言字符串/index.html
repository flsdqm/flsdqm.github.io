2.char指针

明解C语言第十一章会学到，可以通过指针操纵字符串。

指针也就是内存地址，指针变量是用来存放内存地址的变量。地址相当于房号，房间里住的是存放的是某个类型(int, double…)。而指针存的就是房号，指针通过访问房号，来找到房间里的住户。

我们需要知道字符串再内存单元的存放方式通常是一段连续地址。

这里char *是一个类型，代表str是一个指向char类型的指针变量，而这个指针变量str存的地址是字符串”hello”的首地址。

二、一些常用的string.h中的方法

这些方法其实都可以自己用C语言实现，可以尝试一下。

1.strlen

strlen用于计算字符串长度，返回一个无符号整型，长度不包括’\0’。

下面就是strlen用C语言实现的代码。

2.strcpy

char *strcpy(char *dst, char const *src);

strcpy用于复制字符串，将src复制到dst。

3.strcat

char *strcat(char *dst, char const *src);

连接两个字符串，将src连接在dst之后，返回dst。

4.strcmp

int strcmp(char const *s1, char const *s2);

依次比较s1、s2的第一位，第二位……，当有不一样时

1）s1 < s2时，返回一个负数，通常为-1；

2）s1 > s2时，返回一个正数，通常为1；

当s1和s2一样时，返回0。
